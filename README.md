# Bases de Datos SQLite con Python

## Descripción

Repositorio que contiene ejercicios prácticos relacionados con la gestión de bases de datos SQLite utilizando Python.

## Tecnologías utilizadas

- Python 3
- SQLite
- SQL
- Jupyter Notebook

## Conceptos trabajados

- Creación de bases de datos
- Consultas SQL
- Inserción de registros
- Actualización de datos
- Eliminación de registros
- Persistencia de información

## Objetivo

Aprender a almacenar y gestionar información mediante bases de datos SQLite integradas con aplicaciones Python.
